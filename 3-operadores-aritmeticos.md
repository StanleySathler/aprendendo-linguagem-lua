# Guia de Introdução à Linguagem Lua para Crianças

## (3) Operadores Aritméticos

Use operadores para calcular.

- `+` : Soma
- `-` : Subtração
- `*` : Multiplicação
- `/` : Divisão

### Exemplos

```lua
local a = 10
local b = 5

print(a + b)  -- 15
print(a - b)  -- 5
print(a * b)  -- 50
print(a / b)  -- 2
```

### Exercícios

> Quando dizemos "Imprima", significa usar `print()` para mostrar o valor na tela.

1. Some 7 + 3. Imprima o resultado.

2. Subtraia 10 - 4. Imprima o resultado.

3. Multiplique 6 * 2. Imprima o resultado.

4. Divida 20 / 5. Imprima o resultado.
