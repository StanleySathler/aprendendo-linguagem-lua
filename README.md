# Guia de Introdução à Linguagem Lua para Crianças

Um guia para crianças que querem aprender a programar em Lua. Recomendado para adultos também.

## Por que aprender Lua?

Lua é usado em vários jogos, como Roblox e GTA V (FiveM). Aprender Lua pode ser o primeiro passo para criar seus próprios jogos e aplicativos!
