# Guia de Introdução à Linguagem Lua para Crianças

## Variáveis e Tipos

Variáveis guardam valores. Use `local` para criar.

### Tipos

- **String**: Texto, entre aspas. Ex: `"Olá"`
- **Number**: Números. Ex: `10`
- **Boolean**: Verdadeiro ou falso. Ex: `true`, `false`

### Exemplos

```lua
local nome = "João"  -- String
local idade = 10     -- Number
local gosta = true   -- Boolean

print(nome)
print(idade)
print(gosta)
```
