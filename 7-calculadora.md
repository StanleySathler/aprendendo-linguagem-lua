# Guia de Introdução à Linguagem Lua para Crianças

## Exercício prático - Calculadora

Parabéns, futuro programador! Você está fazendo um ótimo progresso.

Programação se aprende praticando. E para praticar ainda mais, vamos construir um pequeno projeto - uma calculadora.

### Instruções

Seu programa deve:

- Ler dois números do teclado.
- Ler a operação: +, -, *, /.
- Se +, imprima a soma.
- Se -, imprima a subtração.
- Se *, imprima a multiplicação.
- Se /, imprima a divisão (cuidado com divisão por zero).

### Exemplo

Digite num1: 10  
Digite num2: 5  
Digite operação: +  
Resultado: 15
