# Guia de Introdução à Linguagem Lua para Crianças

Olá, futuro programador!

Para executar seus programas em Lua, você precisa de:

1. Ter o Lua instalado no seu computador.
2. Ter um editor de códigos instalado no seu computador - ex. Visual Studio Code.
3. Ter um terminal - ex. Prompt de Comando ou PowerShell (Windows).

## Instalando o Lua

### Windows

1. Baixe e instale o Lua para Windows.
   - Acesse a [página de download](https://github.com/rjpcomputing/luaforwindows/releases).
   - Selecione a versão mais recente e baixe o instalador - ex. `LuaForWindows_v5.1.5-52.exe`.

## Criando programas em Lua

Seu primeiro programa é simples.

Crie um arquivo chamado `ola_mundo.lua` com o seguinte conteúdo:

```lua
print("Olá, Mundo!")
```

Salve em algum lugar do seu computador - por ex, área de trabalho.
